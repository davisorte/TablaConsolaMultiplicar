# Mis Notas
Es una aplicacion de node que simula una calculadora en consola
```
     --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Es la base de multiplicar                  [número] [requerido]
  -l, --listar   Muestra la tabla de multiplicar en consola
                                                     [booleano] [defecto: false]
  -h, --hasta    Muestra la tabla de multiplicar en consola
                                                          [número] [defecto: 10]

```
